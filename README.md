# AI Sign Language Translator

An AI-powered communication tool that helps bridge the gap between hearing, deaf, and speech-impaired individuals. This application uses computer vision and machine learning to recognize sign language gestures through a webcam and convert them into readable text in real time.

The system also includes a speech-to-text feature that converts spoken words into text, allowing deaf users to read and understand conversations instantly. By combining sign language recognition and speech transcription, the application creates a more inclusive and accessible communication experience.

## Features

* Real-time sign language recognition using a webcam
* AI-based hand gesture detection and translation
* Conversion of sign language gestures into text
* Speech-to-text functionality for deaf users
* User-friendly and accessible interface
* Real-time communication support

## Purpose
The goal of this project is to make communication more accessible for people with hearing and speech disabilities by providing a real-time AI-powered translation system between sign language, spoken language, and text.
